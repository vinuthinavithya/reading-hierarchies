# Reading Hierarchies

Reading Hierarchies is a digital design project exploring 
Material, Height, and Otherness in Kandyan Architecture.

## Concept

This project studies how:
- Material choices
- Height regulations
- Sacred architecture

function as a silent social language in Kandy.

## Features

- Interactive website
- AI-inspired chatbot
- Audio documentation
- Image galleries
- Scroll animations
- Game exploring Otherness

## Technologies Used

- HTML / CSS
- JavaScript
- PHP
- Rule-based NLP chatbot logic
